## Project Summary
A modern, high-end data science portfolio for Abhirami R, featuring advanced animations, sophisticated typography, and a distinctive dark aesthetic.

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React
- Radix UI (shadcn/ui)

## Architecture
- `src/app/page.tsx`: Main portfolio page
- `src/components`: UI components and section wrappers
- `src/lib`: Utility functions

## User Preferences
- Distinctive typography (Syne & Plus Jakarta Sans)
- High-impact motion and reveals
- Dark theme with sharp neon accents
- No unnecessary comments

## Project Guidelines
- Keep components as Server Components where possible, use 'use client' for interactivity.
- Use CSS variables for consistent theming.
- Implement responsive design for all screen sizes.

## Common Patterns
- Framer Motion for entrance animations (staggered children).
- Glassmorphism for cards and navigation.
